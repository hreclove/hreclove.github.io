기본적인 소프트웨어 설치하기

 

1. 크롬 브라우저 설치하기

* Windows 10/8.1/7

* Mac OS X 10.9 이상

 

2. 크롬 브라우저에서 사용할 확장 기능(Plugin) 설치하기 (택 1)

* 브라우저로 다운로드 후 설치 하는 경우 (Windows, Mac OS)

* 크롬 브라우저에서 직접 설치 하는 경우 (Windows, Mac OS)

 

[주의 사항] 사용자의 Windows OS에 따라서는 관리자 권한으로 크롬 브라우저를 실행해야 만 확장 기능이 동작합니다.

 

설치 환경에 대한 공식 설명 사이트: https://scratch.mit.edu/info/ext_download/
