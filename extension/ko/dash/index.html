Scratch X는 현재 Beta 공개 상태이며, JavaScript 기반으로 동작하므로 브라우저 접속을 하여 실행/편집을 할 수 있습니다.

Scratch X에서 이 확장 기능을 사용하려면 반드시 별도의 장치(USB 동글)가 필요 합니다.

(별도 장치는 추후 공개 예정임)

 

기본적인 소프트웨어 설치하기

 

1. 크롬 브라우저 설치하기

* Windows 10/8.1/8/7

* Mac OS X 10.9 이상

공식 다운로드 경로: https://www.google.com/chrome/

 

2. 크롬 브라우저에서 사용할 확장 기능(Plugin) 설치하기 (택 1)

* 브라우저로 다운로드 후 설치 하는 경우 (Windows, Mac OS)

* 크롬 브라우저에서 직접 설치 하는 경우 (Windows, Mac OS)

 

[주의 사항]

사용자 PC 환경에 따라서는 관리자 권한으로 실행해야 확장 기능이 동작할 수 있습니다.

 

설치 환경에 대한 공식 설명 사이트 참고: https://scratch.mit.edu/info/ext_download/

 

 

 

 

 

장치 드라이버 설치하기

* 사용자의 PC 환경에 따라서는 별도의 드라이버를 설치해야 합니다.

 

USB동글을 PC에 연결한 이후에 장치관리자에 아래와 같은 CP210x 장치가 발견되지 않을 경우에는 장치 Driver를 다운로드 (Windows, Mac OS)해서 설치하세요. (COM번호는 사용자 환경마다 다릅니다.)

 

 

 

연결 상태 확인하기

Scratch X 와 연결되는 확장 프로그램(Plugin)이 동작하지 않는 경우에 빨간색이 표시됩니다.

확장 프로그램이 설치되어 있는지 다시 확인 바랍니다. 확장 프로그램이 설치되어 있다면, 크롬 브라우저를 다시 실행해 주세요.

 

확장 프로그램이 정상적으로 동작이며, USB동글과 DASH의 연결을 확인 중인 상태를 의미 합니다. DASH와 연결된 상태가 아닙니다.

PC에 연결된 USB동글이 정상적으로 연결되었는지 확인 바라며, USB동글이 정상적으로 연결되어 있다면, DASH 켜고 USB동글 근처에 놓아 주시기 바랍니다.

이 상태가 장시간 유지된다면, 크롬 브라우저를 종료시킨 후 다시 실행해 주시기 바랍니다.

 

확장 프로그램이 정상적으로 동작하며, 외부 장치를 통해서 DASH와 통신이 잘 되는 상태 입니다.

 

 

DASH 제어 기능 사용하기

 

1. 이동하기

 

이동/정지 하거나 회전하는 방향을 지정합니다. 사용되는 값은 초당 움직이는 거리와 방향을 의미합니다.

즉, 이동 속도와 회전 속도를 의미 합니다.

[주의사항] 한번 설정된 값은 “정지” 명령이 전달하기 전까지는 계속 이동/회전 합니다. (실행예제)

 

[회전 속도]

1초에 몇 도를 회전하는지를 지정합니다. (값 범위: -389 ~ 389)

ü  왼쪽 방향 à 1 ~ 389

ü  오른쪽 방향으로 회전 à -389 ~ -1

ü  회전 없음 à 0

 

[이동 속도]

1초에 몇 센티미터를 이동하는지 지정합니다. (값 범위: -100 ~ 100)

ü  정면 방향 à 1~100

ü  뒤면 방향 à -100 ~ -1

ü  정지 à 0

 

[방향]

진행 방향 종류를 지정합니다. (값 범위: 앞으로, 뒤로, 왼쪽, 오른쪽)

ü  앞으로 à 회전 값 0

ü  뒤로 à 회전 값 0, 이동 값을 –로 적용함.

ü  왼쪽 à 회전 값 90

ü  오른쪽 à 회전 값 -90

 

 

2. 머리 방향 돌리기

 

머리부분에 있는 얼굴(눈)이 보게 되는 방향(각도)을 지정합니다. (실행예제)

 

[좌우]

머리가 위치할 방향의 좌우(X) 각도를 지정합니다. (값 범위: -120 ~ 120)

ü  왼쪽 à 1 ~ 120

ü  오른쪽 à -120 ~ -1

ü  정면 à 0

 

[상하]

머리가 위치할 방향의 상하(Y) 각도를 지정합니다. (값 범위: -12 ~ 7)

 

[방향]

머리가 위치할 방향을 지정 합니다. (값 범위: 왼쪽, 오른쪽, 정면, 위, 아래)

ü  왼쪽 à 좌우 90, 상하 0

ü  오른쪽 à 좌우 -90, 상하 0

ü  위 à 좌우 0, 상하 -12

ü  아래 à 좌우 0, 상하 7

ü  정면 à 좌우 0, 상하 0

 

 

3. 조명색 바꾸기

 

LED조명의 색을 바꾸거나, 램프의 밝기와 켜고 끄기를 할 수 있습니다. (실행예제)

 

[LED색 바꾸기]

ü  위치 à 왼쪽 귀, 오른쪽 귀, 가슴

ü  색깔 à 빨강, 주황, 노랑, 초록, 하늘, 파랑, 보라, 흰색, 끄기

 

[LED색 조정하기]

색깔의 빨강, 초록, 파랑 값을 조합하여 색을 만들 수 있습니다. (색깔 값 범위: 0~255)

ü  빨강 à 255, 0, 0

ü  주황 à 255, 128, 0

ü  노랑 à 255, 255 0

ü  초록 à 0, 255, 0

ü  하늘 à 0, 128, 255

ü  파랑 à 0, 0, 255

ü  보라 à 255, 0, 255

ü  흰색 à 255, 255, 255

ü  끄기 à 0, 0, 0

 

[램프 조정]

램프의 켜고 끄기 뿐만 아니라, 밝기를 조정합니다.

ü  위치 à 꼬리 등, 큰 버튼(머리 윗부분),

ü  밝기 à 0 ~ 255 (0: 끄기)

 

 

4. 눈 조명 바꾸기

머리의 정면에 있는 12개의 LED를 켜고 끌 수 있습니다. (실행예제)

 

[눈 조명 제어]

눈 모양에 배치된 LED는 단색으로 시계 시침의 위치와 동일 하며, 방향에 맞춰서 끄고 켤 수 있습니다.

ü  조명 번호 à 1 ~ 12

ü  값 à 0(끄기) 또는 255(켜기)

 

[눈 조명 모양 값]

눈 모양을 한꺼번에 바꿀 때 사용하는 기능으로, 16진수의 값을 그대로 사용 할 수 있습니다. (값 범위: 0x0000 ~ 0x0FFF)

ü  모두 끄기 à 0000

ü  모두 켜기 à 0FFF

5. 소리 재생

내장된 소리를 재생합니다. 소리 명령을 연속으로 전달 시에는 이전 소리가 중지되고 새로 전달된 마지막 소리 명령이 재생 됩니다. (실행예제)

2가지 종류(느낌 표현, 효과음)의 소리를 선택할 수 있으며, 각 소리의 크기는 음량 값으로 변경합니다. (값 범위: 0 ~ 100)

 

 

6. 버튼 감지 하기

버튼의 눌림 여부를 감지하는 기능을 켜고 끌 수 있으며, 버튼의 눌림 상태를 읽거나 이벤트 동작으로 사용 가능합니다. (실행예제)

 

[버튼 감지 설정]

버튼 감지 “끄기” 또는 “켜기”를 선택합니다. 전체 버튼의 눌림 여부를 감지합니다.

버튼 감지는 사용하지 않는 경우에는 “끄기”를 선택하여 동작을 중지 해줘야 합니다.

 

[버튼 감지 조건]

머리 부분의 윗부분에 4개의 버튼이 있으며, 큰 버튼, 1, 2, 3 각각의 눌림 여부를 인식 할 수 있습니다.

 

[버튼 감지 이벤트]

이벤트가 동작하기 위해서는 버튼감지기능이 켜져 있어야 합니다.

 

 

7. 거리 감지 하기

왼쪽 정면, 오른쪽 정면, 뒷면에 있는 거리 감지 센서의 기능을 켜고 끌 수 있으며, 감지된 값을 읽을 수 있습니다. (실행예제)

 

[거리 감지 설정]

거리 감지 “끄기” 또는 “켜기”를 선택 합니다. 3가지 거리 센서를 모두 감시 합니다.

거리 감지를 사용하지 않는 경우에는 “끄기”를 선택하여 동작을 중지 해줘야 합니다.

 

[거리 감지 값]

일정 범위에 장애물이 감지되면 신호의 강도를 값으로 인식 합니다. (값 범위: 0 ~ 255)

(주의) 주변의 조명 상태 및 장애물의 표면 상태에 따라서 감지되는 값이 달라질 수 있습니다.

또한, 매우 근접하면 최대값인 255보다 작은 값이 읽힐 수 있습니다.
